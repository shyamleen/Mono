Lesson 1.4: Double-Entry Matrix Rules

Under double-entry bookkeeping, total accounting transactions must maintain net equal status across systems. You cannot execute a debit entry without logging an offset matching credit entry.

Account Category Debit (Left Side) Effect Credit (Right Side) Effect
Assets ➕ Increases Value ➖ Decreases Value
Liabilities ➖ Decreases Value ➕ Increases Value
